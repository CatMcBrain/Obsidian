
Commands:
```
printf("Print text and %d variables \n", var);

printf("Input your info: ");

printf("Input info for a string of 30: ");
fgets()
```

## Variables and Format specifiers

```
int = whole number (%d)
float = decimal number (%f)
double = long float ($lf)
char = 'single character' (%c)
char[] = "string" (%s)
bool = true/false
```
